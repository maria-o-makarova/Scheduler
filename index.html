connect to firebase

Paste your Firebase config below to sync tasks across devices.

  1. Go to console.firebase.google.com → open your project
  2. Click Project Settings (gear icon) → Your apps → add a Web app
  3. Copy the firebaseConfig object and paste below
  4. Sidebar → Build → Realtime Database → create in test mode
Config stored only in this browser's localStorage.

paste task list

Paste your list — one task per line. Lines starting with ▢ or - or plain text are imported as tasks. Lines like "Weeks X" or notes are skipped automatically. Tasks containing Test, Exam, Quiz, or Midterm get their own exclusive day when re-balancing.

Imported tasks are added to your existing list — duplicates by name are skipped.
